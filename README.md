Tricordarr
==========

A [Twitarr](https://github.com/jocosocial/swiftarr) client.

Quick Start
-----------

```bash
nvm install lts/jod
nvm use lts/jod (v22.13.0)

npm clean-install

# Android (Mac, Windows, Linux)
npx react-native run-android

# iOS (Mac)
npx react-native run-ios
```

See [Docs](./docs/) for more.
